The website features a bold dark-theme aesthetic, smooth scroll animations, and a responsive mobile-first design to drive gym memberships and inquiries.

## ✨ Key Features
* **Modern Dark UI:** Deep blacks and charcoals with fiery orange/gold accent gradients.
* **Scroll Animations:** Smooth fade-ins, floating logo animations, and word-by-word quote reveals using `IntersectionObserver`.
* **Dynamic Counters:** Animated JavaScript number counters for gym statistics (Members, Reviews, Rating).
* **Infinite Marquee:** Auto-scrolling 5-star testimonial carousel.
* **Pricing Cards:** Interactive membership plans with hover-lift effects.
* **Fully Responsive:** Customized mobile hamburger menu and stacked grid layouts for smaller screens.
* **Instant Contact:** Floating WhatsApp pulse button and one-click "Call Now" actions linked to the gym's mobile number.

## 🛠️ Tech Stack
* **HTML5:** Semantic markup structure all contained in a single `index.html` file.
* **Tailwind CSS (via CDN):** Rapid utility-first styling and responsive breakpoints.
* **Vanilla JavaScript:** For mobile menu toggling, scroll-triggered animations, and number counters (No external JS libraries needed).
* **Google Fonts:** `Bebas Neue` for impactful headings and `Poppins` for clean body typography.
* **Inline SVGs:** Auto-generated vector graphics for the logo to ensure crisp scaling without external image dependencies.

## 🚀 How to Run Locally
Because this project is built entirely in a single file with CDN links, running it is incredibly simple:
1. Clone this repository or download the ZIP file.
2. Open the folder on your computer.
3. Double-click the `index.html` file to open it in any web browser (Chrome, Firefox, Safari, Edge).
